# Continuous Learning

## What to Extract from Sessions

### Successful Patterns
- Effective prompt structures
- Good agent delegation sequences
- Time-saving workflows
- Problem-solving approaches that worked

### Mistakes & Lessons
- Ambiguous instructions that caused confusion
- Missing context that led to wrong output
- Over-engineered solutions
- Tasks that needed more/less delegation

### Knowledge to Persist
- Project-specific conventions discovered
- Useful terminal commands and flags
- Configuration gotchas and workarounds
- Library/API quirks found

## Learning Loop
```
Execute task → Observe outcome → Extract patterns
     ↑                                   ↓
     └──── Apply improved approach ←─────┘
```

## Skill Evolution
- Clusters of related patterns → new skill
- Outdated patterns → update or remove
- Frequently used workflows → command shortcut

## When to Learn
- After major feature completion
- After fixing a hard bug
- After discovering a new pattern
- When onboarding new project members
