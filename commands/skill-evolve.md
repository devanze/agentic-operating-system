Create a skill from an instinct cluster. Run `node .opencode/scripts/instinct.js suggest <clusterId>` to get a skill template.

Generate a complete SKILL.md file from the clustered instincts:
1. Review the cluster's instincts
2. Create a skill name from the cluster category
3. Write the SKILL.md with proper frontmatter (name, description)
4. Include all the instinct content as the skill body
5. Place in `skills/<name>/SKILL.md`

This turns learned patterns into reusable, auto-loaded knowledge.
