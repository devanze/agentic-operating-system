Promote project instincts to the global scope so they apply across all projects.

Options:
- `/promote <instinct-id>` — Promote a single instinct to global
- `/promote cluster <cluster-id>` — Promote an entire cluster to global

Run `node .opencode/scripts/instinct.js promote <id>` or `node .opencode/scripts/instinct.js promote-cluster <clusterId>`.

Global instincts are available to ALL projects, not just this one.
