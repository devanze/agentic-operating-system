Evolve instincts into clusters ready for skill creation.

Run `node .opencode/scripts/instinct.js evolve [minCount]` to cluster similar instincts together.

- Default: cluster instincts appearing 3+ times with similar titles
- Clusters become candidates for new skill packs
- Review clusters with `/instinct-status`
- Promote good clusters to global with `/promote`
- Generate skills from clusters with `/skill-create`
